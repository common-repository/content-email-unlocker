Display content after subscribe to newsletters

== Description ==
Content Email Unlocker allows you to block content for people who are not subscribed to newsletter. To unlock content the user has to enter their email address.

You can set the confirmation e-mail address. If user is already registered, the content will show without sending confirmation.

Go to settings and select a newsletter system, and a list to which should be saved.

Confirmation email is sent by the mailing system, so you also need to have properly configured newsletter.

Clicking on the confirmation link will be redirected to the page with the article.

Locked content is displayed by AJAX method.

The plugin uses cookies to store an access.

How to use?

Just insert shortcode like [emailunlock]my text to lock[/emailunlock]

== Installation ==
1. Upload the plugin to the /wp-content/plugins/ directory or install via wordpress.
2. Activate the plugin through the \'Plugins\' menu in WordPress
3. Go to wordpress Settings->Content Email Unlocker. Select Newsletter System. Go to your post and add shortcode [emailunlock]text[/emailunlock] to your content.
